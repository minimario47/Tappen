Tåppen – Integritetspolicy

1. Inledning

Välkommen till Tåppens integritetspolicy. Här beskriver vi hur vi hanterar och skyddar din personliga information när du använder vår tågapp.

2. Användning av Platsdata

Tåppen kan använda din enhets platsinformation för att visa din aktuella position på kartan och underlätta din resa. Appen ber om tillstånd innan platsdata samlas in, och du kan när som helst återkalla detta tillstånd via din enhets inställningar.

Varför vi använder platsdata:

  • För att visa din nuvarande position på kartan.
  • För att föreslå närmaste station, plattform eller spår.

Ingen platsdata skickas till en tredjepart eller lagras.

3. Ingen Insamling av Övrig Personlig Data

Utöver platsdata samlar Tåppen inte in personligt identifierbar information, som namn eller e-post, utan din uttryckliga tillåtelse.

4. Delning av Data

Vi delar inte din plats eller annan användningsdata och kan heller inte se sådan data.

6. Ändringar i Policyn

Vi kan komma att uppdatera denna integritetspolicy vid behov. Vi rekommenderar därför att du regelbundet besöker denna sida för att hålla dig uppdaterad om eventuella förändringar.
